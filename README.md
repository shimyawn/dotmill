# DOTMILL Guestbook

닷밀 신년맞이 방명록 웹앱입니다.

"dotmill" 텍스트가 물리 시뮬레이션 도트로 표현되며, 방문자가 메시지를 남기면 포스트잇 형태로 표시됩니다.
아바타를 커스터마이징하고, 다양한 인터랙션 효과를 체험할 수 있습니다.

## Features

- 물리 시뮬레이션 기반 도트 타이포그래피 ("dotmill")
- 포스트잇 방명록 (메시지 작성/조회)
- 아바타 커스터마이징 (얼굴, 머리, 색상)
- 인터랙티브 효과 (도트 쏟아지기, 산타 모드 등)
- Director's Note
- 튜토리얼 가이드
- 캐럴 BGM + 효과음

## Tech

- Vanilla JavaScript + HTML5 Canvas
- Google Apps Script 백엔드 (메시지 저장/조회)
- 단일 파일 (`index.html`)
- 외부 라이브러리 없음
